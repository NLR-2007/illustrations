# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-14

### Added
- Core modular layout engine supporting hierarchical flowcharts, swimlane roadmaps, horizontal timelines, Venn diagrams, and side-by-side comparison tables.
- Editable PowerPoint deck builder integrated with `PptxGenJS`.
- Automated text-fitting algorithms (resizing fonts and shifting nodes).
- AI Visual Router for query mode decision-making.
- Prompt consistency engine (locks character and style rules).
- Multi-provider abstraction (`manual`, `openai-compatible`, `custom-http`).
- Validation system auditing overlaps, margins, and prompts.
- Interactive CLI executable scripts.
- Complete Vitest unit testing coverage.
