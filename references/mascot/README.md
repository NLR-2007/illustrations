# Mascot Reference Images

Place your mascot reference images in this directory. 

The mascot visual identity:
- Simple rounded white body
- Large expressive black eyes
- Small friendly mouth
- Thin imperfect hand-drawn black arms
- Thin imperfect hand-drawn black legs
- Two small bright yellow chest marks
- White background, line art with yellow accents

These images will be analyzed or referenced when generating scenes in AI image generation systems.
