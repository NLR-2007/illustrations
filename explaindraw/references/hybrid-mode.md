# Hybrid mode

1. Use hybrid mode only when illustration and structure each communicate different necessary information.
2. Divide a 16:9 canvas into non-overlapping illustration and diagram regions, normally 40%/60% or 45%/55%.
3. Let the mascot scene introduce the metaphor or actor. Let the diagram explain the exact flow, sequence, or system.
4. Use `assets/examples/hybrid.png` as composition calibration.
5. Keep one shared title, palette, baseline, and line character across both regions.
6. Generate the illustration with the image tool. Build diagram elements as native editable shapes when presentation tools exist.
7. Do not merge both sides into an uneditable bitmap when the user asked for PowerPoint editability.
8. Confirm that neither pane crosses the other pane's boundary and that labels remain legible at slide scale.
