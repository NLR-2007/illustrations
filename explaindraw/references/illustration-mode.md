# Illustration mode

1. Identify one concrete metaphor or physical action that explains the topic.
2. Make the mascot participate in the core action rather than stand as decoration.
3. Limit the scene to the mascot, two to five essential objects, and at most four directional cues.
4. Use `assets/mascot/` as character references and `assets/examples/illustration.png` as density and spacing calibration.
5. Generate the actual bitmap with the host image tool. Default to 16:9, pure white, black hand-drawn linework, and yellow accents only.
6. Avoid paragraphs and unnecessary labels. Prefer recognizable objects and arrows.

For image tools supporting a negative prompt, exclude photorealism, 3D, gradients, shadows, scenery, extra colors, extra limbs, malformed hands, clothing, watermarks, signatures, borders, and unintended text.
