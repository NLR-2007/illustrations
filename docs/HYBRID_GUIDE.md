# Hybrid Mode Guide

Hybrid Mode brings together the emotional, storytelling strength of mascot illustrations and the structured, clear hierarchy of editable PowerPoint diagrams.

---

## 1. Supported Templates

The system supports eight hybrid templates, arranged on a widescreen canvas.

```
+------------------------------------+
|               TITLE                |
+------------------+-----------------+
|                  |                 |
|   ILLUSTRATION   |    EDITABLE     |
|   (Mascot/Story) |    DIAGRAM      |
|                  |                 |
+------------------+-----------------+
            Illustration Left + Diagram Right
```

### 1. Illustration Left + Diagram Right
- **Left Side**: Placed Mascot-based illustration explaining the analogy or concept.
- **Right Side**: Native editable flowchart or block diagram showing the actual system pipeline.

### 2. Diagram Left + Illustration Right
- Mirror of template 1. Used for contrasting steps or highlighting the structural flow before zooming into the emotional scenario.

### 3. Illustration Top + Process Bottom
- **Top Half**: High-level conceptual story illustration.
- **Bottom Half**: Horizontal editable timeline or process roadmap.

### 4. Central Illustration + Supporting Shapes
- Mascot is placed in the center, with editable text callouts, shapes, and arrows pointing outward to explain subsystems.

### 5. Full-width Explanation
- A top header, a middle visual explanation combining small mascot blocks, and a bottom summary table.

### 6. Story + Flow
- Left: Short narrative text blocks with a small mascot icon.
- Right: Detailed system flow with decision trees.

### 7. Problem + Solution
- Left (Problem): Sad/confused mascot illustration with text.
- Right (Solution): Happy/efficient mascot illustration with an editable success path diagram.

### 8. Before + After
- Left (Before): Chaotic diagram/illustration.
- Right (After): Clean, optimized diagram/illustration highlighted in accent yellow.

---

## 2. Layout Dimensions (16:9)

On a widescreen slide (`13.33 x 7.5` inches):

- **Header Title Area**: `0.0 to 1.5` inches (vertical).
- **Split Coordinates (50/50 vertical division)**:
  - Left pane: `X: 0.8, Y: 1.8, W: 5.5, H: 4.8` inches.
  - Right pane: `X: 7.0, Y: 1.8, W: 5.5, H: 4.8` inches.
- **Split Coordinates (Horizontal division)**:
  - Top pane: `X: 0.8, Y: 1.8, W: 11.7, H: 2.2` inches.
  - Bottom pane: `X: 0.8, Y: 4.3, W: 11.7, H: 2.3` inches.

---

## 3. Integration Rules
1. **Never Flatten the Diagram**: The diagram elements (rectangles, diamonds, connectors, text) must remain editable PowerPoint components. Only the illustration should be placed as an image.
2. **Harmonious Styling**: The colors of the diagram shapes must strictly match the colors of the illustration (black outlines, yellow accents, white backgrounds).
3. **Alignment**: Keep the text alignment and spacing of the left and right sides balanced.
